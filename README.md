# Cellular Automata: From Conway's Life to Physical Systems

This repository contains the materials for an elective course I authored and taught at the **Krasnoyarsk Summer School (KLSH)** for gifted high school students.

The course bridges the gap between discrete mathematics, computer science, and computational physics, focusing on how simple local rules lead to complex emergent behavior.

---

## 📚 Course Curriculum

The course is structured into 5 modules, each containing a theoretical lecture and a Python-based practical workshop:

1. **Introduction to CA:** Formal definition, Wolfram’s 1D elementary automata, remembering the basics of Python.
2. **Realisation** 1D automata, pattern classifications (stable, chaotic, periodic).
3. **Game of Life and Beyond:** 2D automata, the logic of moving to a higher dimension, and pattern classifications (stills, oscillators, spaceships).
4. **Object-oriented programming:** hte basics and implementation in our game.
5. **Physical Systems Modeling:** 
    - **Diffusion** Using CA for heat equations and snowflakes.

---

## 🛠 Tech Stack & Implementation

- **Language:** Python 3.10+
- **Core Libraries:** `Pygame` (for animations), `Matplotlib` (for static images and animation).
- **Concepts:** Discrete Dynamical Systems, Complexity Theory, Computational Physics.

## 👨‍🏫 Pedagogical Approach

As a student of **MIPT (Phystech)**, I designed this course to maintain a balance between academic rigor and intuitive understanding. My goal was to show students how computational methods can solve problems that are analytically intractable.
